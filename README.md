# Pass / Fail Predictor (Machine Learning)

## Problem Chosen
Pass / Fail Predictor using Machine Learning Classification.

## Dataset Source
Student Performance Dataset (Kaggle)  
https://www.kaggle.com/datasets/spscientist/students-performance-in-exams

## Description
This project predicts whether a student will pass or fail based on their
Math, Reading, and Writing scores using a Logistic Regression model.

Pass criteria:
- Pass if average score >= 40
- Fail otherwise

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the project:
   python predict.py

3. Enter student marks when prompted.

## Output
The model displays accuracy and predicts PASS or FAIL based on input scores.
