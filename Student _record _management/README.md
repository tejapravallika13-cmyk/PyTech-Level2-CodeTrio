# Student Record Management System

## Problem Chosen
Student Record Management System (CRUD)

This project is a terminal-based application that allows users to manage student records efficiently.
It follows a clean modular structure using object-oriented programming.

## Features

- Add new student
- View all students
- Update student details 
- Delete student
- Data persistence using `data.txt`
- Proper input validation
- Error handling for invalid inputs

## Technologies Used

- Python 3
- Built-in Python libraries only
- File Handling
- Object-Oriented Programming (OOP)

##  Project Structure

StudentRecord/
│
├── main.py 
├── model.py 
├── data.txt 
└── README.md 

##  How to Run the Project

1. Open terminal
2. Navigate to project folder
3. Run: python main.py

The program runs fully in the terminal.
To exit the program, press `Ctrl + C`.

No external dataset is used.
